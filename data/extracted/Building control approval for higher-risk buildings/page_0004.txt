When to get building control approval
You must get building control approval from BSR before starting:
● construction of a new higher-risk building
● building work to an existing building that either makes it a higher-risk building, or stops it
being a higher-risk building
● building work to an existing higher-risk building, unless it consists only of exempt work,
competent person scheme work or emergency repairs
Carrying out building work without the required building control approval is a criminal
offence.
You may also need to get planning permission before applying for building control approval. If you’re
not sure, check with the local authority.
Transitional buildings
If the building work started before BSR became the building control authority, transitional
arrangements may apply.
You can find out more by reading the guidance about:
● transitional building control approval for new higher-risk buildings
● transitional building control approval for work on existing higher-risk buildings
Building work that does not need building control approval
You do not need to apply for building control approval for work which consists only of:
● exempt work
● competent person scheme work
