Applying to construct a new higher-risk building
How to apply: new higher-risk buildings
The client or someone authorised to act on their behalf should submit and manage the building
control application online.
You can read the guidance about preparing information for a building control approval application. It
describes the information and documents you’ll need to submit for your application.
Check how many applications you need: new higher-risk buildings
You can submit a single application for a:
● single tower project, which is any higher-risk building that is not connected to another
building
● multiple tower project, this includes multiple higher-risk buildings connected to each other
by a shared podium or common part, such as a car park, walkway or party wall
Staged applications
You can apply for a ‘staged application’. This allows you to:
● progress the project in separate stages
● submit detailed plans for each stage prior to being able to start work on each stage
You can apply for a staged application for:
● single tower projects
● multiple tower projects
