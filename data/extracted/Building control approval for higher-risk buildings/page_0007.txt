Before submitting a staged application for a multiple tower project, you should contact BSR for
advice.
You must not start any building work until the building control application for that stage is
approved.
BSR reviews staged applications on a case-by-case basis.
Applying to carry out building work on an existing
higher-risk building
Before you apply, check that you need to submit an application. Do not submit a building control
approval application for work that consists only of exempt work, competent person scheme work or
emergency repairs.
How to apply: existing higher-risk buildings
The client or someone authorised to act on their behalf should submit and manage the building
control application online.
You can read the guidance about preparing information for a building control approval application. It
describes the information and documents you’ll need to submit for your application.
If you need help with your application, you can get advice from a competent contractor or other
building control professional.
