You can apply for a ‘staged application’. A staged application is where building work is split into
different stages and each stage:
● is treated and assessed separately
● can start or finish on the same date, or different dates, as other stages
You can apply for a staged application where stage one of the project meets the definition of
‘commencement’. This is different for complex and non-complex buildings.
This guidance explains staged applications for:
● multiple tower projects in complex buildings
● single tower projects with no more than one storey below ground level, in
non-complex buildings
Staged applications for multiple tower projects
A staged application may be appropriate for a multiple tower project where:
● buildings are on a shared podium with multiple connected parts
● as a minimum, all foundations and structure of the stage one tower’s lowest floor level
will be completed
Subsequent stages should relate to the remaining building work for the towers.
Before submitting a staged application for a multiple tower project, you should contact
BSR for advice.
Staged applications for single tower projects
A staged application may be appropriate for a single tower project which has no more than one
storey below ground level, where the following will be completed in stage one:
● the foundations
● any basement level
